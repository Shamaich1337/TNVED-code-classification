Description\n7.1 Overview\nThe LM317 isa500-mA linear regulator with high voltage tolerance upto40-V. The device has afeedback\nvoltage that isrelative totheoutput instead ofground. This ungrounded design allows theLM317 tohave\nsuperior lineandload regulation. Italso allows theLM317 tobeused asacurrent source orcurrent sink using a\nsingle resistor. Any output voltage output voltage from 1.25-V to37-V can beobtained byusing tworesistors.\nThe bias current ofthedevice, upto10-mA, flows totheoutput; thiscurrent must beused bytheload orthe\nfeedback resistors. The power dissipation will bethe product ofpass element voltage and current,\nmathematically:\n(V[input] –V[output]) ×I[output] (1)\nThe application heat sink must beable toabsorb thispower.\n7.2 Functional Block Diagram\n7.3 Feature Description\n7.3.1 NPN Darlington Output Drive\nNPN Darlington output topology provides naturally lowoutput impedance and anoutput capacitor isoptional. To\nsupport maximum current andlowest temperature, 2.5-V headroom isrecommended (VI–VO).\n7.3.2 Overload Block\nCurrent limit scales with higher voltage toprovide asafe operation area fortheNPN pass elements. Over\ntemperature shutdown protects against overload orunder heat sinking.\n7.3.3 Programmable Feedback\nOpamp with 1.25-V offset input attheADJUST terminal provides easy output voltage orcurrent (not both)\nprogramming. Forcurrent regulation applications, asingle resistor whose resistance value is1.25V/IOUT and\npower rating isgreater than (1.25V)2/Rshould beused. Forvoltage regulation applications, tworesistors setthe\noutput voltage. See theTypical Application section forschematic andresistor formula.\nCopyright ©2000 –2014, Texas Instruments Incorporated Submit Documentation Feedback 7\nProduct Folder Links: LM317M\nLM317M\nSLVS297P –APRIL 2000 –REVISED MARCH 2014 www.ti.com\nFeature Description (continued)\n7.3.4 Current Sources\nThe two current sources represent the typical ‘adjust terminal current ’and typical ‘minimum current for\nregulation\'.\n7.4 Device Functional Modes\n7.4.1 Normal operation\nThe device OUTPUT terminal willsource current necessary tomake OUTPUT terminal 1.25-V greater than\nADJUST terminal toprovide output regulation.\n7.4.2 Operation With Low Input Voltage\nThe device requires upto2.5-V headroom (Vin-Vout) tooperate inregulation. ### Component Summary: LM317MQDCYR

**Manufacturer:** Texas Instruments  
**Product Code:** LM317MQDCYR  
**Type:** Adjustable Voltage Regulator

#### Key Specifications:
- **Voltage Ratings:**
  - Input Voltage Range: 2.5 V to 40 V (input-to-output differential)
  - Output Voltage Range: 1.25 V to 37 V
- **Current Ratings:**
  - Output Current: Greater than 500 mA
- **Power Consumption:**
  - Quiescent Current: 50 µA to 100 µA
- **Operating Temperature Range:**
  - Junction Temperature (TJ): -40 °C to 125 °C
- **Package Type:**
  - SOT-223 (DCY)
- **Special Features:**
  - Internal short-circuit current limiting
  - Thermal overload protection
  - Safe operating area compensation
  - Adjustable output voltage with two external resistors
- **Moisture Sensitive Level (MSL):**
  - Level 2, according to JEDEC J-STD-020E

#### Description:
The LM317M is a 3-terminal adjustable linear voltage regulator designed to provide a regulated output voltage that can be set between 1.25 V and 37 V. It is capable of supplying output currents greater than 500 mA and features built-in protection mechanisms such as short-circuit current limiting and thermal overload protection. The device operates with a maximum input-to-output differential voltage of 40 V, making it suitable for a variety of applications where adjustable voltage regulation is required.

#### Typical Applications:
- **Power Management:** The LM317M is commonly used in power supply circuits to provide stable voltage levels for various electronic devices.
- **Electronic Point-of-Sales Systems:** It can be utilized to ensure that sensitive electronics receive the correct voltage.
- **Medical/Health/Fitness Applications:** The regulator is suitable for powering medical devices that require precise voltage regulation.
- **Printers and Appliances:** It is often found in consumer electronics and appliances where adjustable voltage is necessary.
- **TV/Set-Top Boxes:** The LM317M can be used to regulate voltage in entertainment devices.

This component is particularly valued for its ease of use, requiring only two external resistors to set the desired output voltage, and its ability to maintain regulation under varying load conditions.
