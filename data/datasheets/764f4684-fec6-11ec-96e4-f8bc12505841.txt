This reduces the overall\nsystem power consumption as the host processor does not need to continuously poll data from the sensor but\ncan go into standby mode for longer periods of time.\nThe FIFO can store any combination of pressure and temperature results, according to the background mode\nmeasurement rate settings. The pressure rate can for instance be set 4 times higher than the temperature rate\nand thus only every fifth  result will be a temperature result. The measurement type can be seen in the result\ndata. The sensor will set the least significant bit to:\n• \'1\' if the result is a pressure measurement.\n• \'0\' if it is a temperature measurement.\n- The sensor uses 24 bits to store the measurement result. Because this is more bits than is needed\nto cover the full dynamic range of the pressure sensor, using the least significant bit to label the\nmeasurement type will not affect  the precision of the result.\nThe FIFO can be enabled in the Interrupt and FIFO configuration register. The data from the FIFO is read out\nfrom the Pressure Data (PRS_Bn) registers regardless of whether the next result in the FIFO is a temperature or a\npressure measurement.\nWhen a measurement has been read out, the FIFO will auto increment and place the next result in the data\nregister. A flag will be set in the FIFO Status register when the FIFO is empty and all following reads will return\n0x800000.\nIf the FIFO is full, the FIFO_FULL bit in the FIFO Status (FIFO_STS)will be set. ### Component Summary: DPS310XTSA1

**Description:**
The DPS310 is a digital barometric pressure sensor designed for portable devices. It features high accuracy and low current consumption, capable of measuring both pressure and temperature. The sensor utilizes a capacitive sensing principle for precise measurements, even with temperature variations. It is particularly suited for mobile applications and wearable devices, providing a compact solution for pressure and temperature sensing.

**Key Specifications:**
- **Voltage Ratings:**
  - Supply Voltage (VDD): 1.7 – 3.6 V
  - Digital Supply Voltage (VDDIO): 1.2 – 3.6 V

- **Current Ratings:**
  - Average Current Consumption: 
    - Pressure Measurement: 1.7 µA (at 1 Hz sampling rate)
    - Temperature Measurement: 1.5 µA (at 1 Hz sampling rate)
    - Standby: 0.5 µA
  - Peak Current Consumption: 
    - 345 µA during pressure measurement
    - 280 µA during temperature measurement

- **Power Consumption:**
  - Varies based on measurement mode and precision settings, with typical values around 2.1 µA for low precision and up to 200 µA for high precision in background mode.

- **Operating Temperature Range:**
  - -40 °C to 85 °C

- **Package Type:**
  - 8-pin LGA (2.0 mm x 2.5 mm x 1.0 mm)

- **Special Features:**
  - High precision pressure sensor with ±0.002 hPa accuracy in high precision mode.
  - FIFO buffer capable of storing up to 32 measurement results.
  - Supports I2C and SPI interfaces for communication.
  - Individually calibrated with stored calibration coefficients for enhanced accuracy.
  - Green Product (RoHS) compliant.

- **Moisture Sensitive Level (MSL):**
  - Rated as MSL1 according to IPC/JEDEC J-STD-020E.

**Typical Applications:**
- **Indoor Navigation:** Used for floor detection in environments like shopping malls and parking garages.
- **Health and Sports:** Provides accurate elevation gain and vertical speed measurements.
- **Outdoor Navigation:** Enhances GPS accuracy and start-up time, particularly in challenging environments like tunnels.
- **Weather Stations:** Facilitates local weather forecasting and micro-weather monitoring.
- **HDD Drivers:** Assists in leak rate detection in hard disk drives.
- **Drones:** Contributes to flight stability and height control.

This summary encapsulates the essential characteristics and applications of the DPS310XTSA1, making it a versatile choice for various pressure and temperature sensing needs in portable devices.
