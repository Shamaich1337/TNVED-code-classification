These\nµP supervisory circuits assert reset to prevent codeexecution errors during power-up, power-down, orbrownout conditions.\nReset asserts when V\nCCis below the reset threshold;\nonce V CCexceeds the reset threshold, an internal timer\nkeeps the reset output asserted for the reset timeoutperiod. After this interval, reset output deasserts. Resetoutput is guaranteed to be in the correct logic state forV\nCC≥1V.\nManual Reset Input (MAX6384/\nMAX6385/MAX6386/MAX6390)\nMany µP-based products require manual reset capabil-\nity, allowing the operator, a test technician, or externallogic circuitry to initiate a reset. A logic low on MR\nasserts reset. Reset remains asserted while MRis low,\nand for the reset active timeout period (t\nRP) after MR\nreturns high. This input has an internal 63k Ωpullup\nresistor (1.56k Ωfor MAX6390), so it can be left uncon-\nnected if it is not used. MRcan be driven with TTL or\nCMOS logic levels, or with open-drain/collector outputs.Connect a normally open momentary switch from MRto\nGND to create a manual-reset function; externaldebounce circuitry is not required. If MRis driven from\nlong cables or if the device is used in a noisy environ-ment, connecting a 0.1µF capacitor from MRto GND\nprovides additional noise immunity. \nRESET IN Comparator\n(MAX6387/MAX6388/MAX6389)\nRESET IN is compared to an internal +1.27V reference.\nIf the voltage at RESET IN is less than 1.27V, resetasserts. Use the RESET IN comparator as a user-adjustable reset detector or as a secondary power-sup-ply monitor by implementing a resistor-divider at RESETIN (shown in Figure 1). Reset asserts when either V\nCC\nor RESET IN falls below its respective threshold volt-age. Use the following equation to set the threshold:\nV\nINTH = VTHRST (R1/R2 + 1)\nwhere V THRST = +1.27V.  To simplify the resistor selec-\ntion, choose a value of R2 and calculate R1:\nR1 = R2 [(V INTH/VTHRST ) - 1]\nSince the input current at RESET IN is 50nA (max),\nlarge values can be used for R2 with no significant lossin accuracy.  ___________Applications Information\nNegative-Going V CCTransients\nIn addition to issuing a reset to the µP during power-up,power-down, and brownout conditions, theMAX6381–MAX6390 are relatively immune to short dura-tion negative-going V\nCCtransients (glitches).\nThe Typical Operating Characteristics section shows the\nMaximum Transient Durations vs. Reset ComparatorOverdrive, for which the MAX6381–MAX6390 do notgenerate a reset pulse. ### Component Summary: MAX6390XS29D4+T

#### Key Specifications:
- **Voltage Ratings**: 
  - Operating Voltage Range: 1.0V to 5.5V
  - Reset Thresholds: Available from +1.58V to +4.63V in approximately 100mV increments.
  
- **Current Ratings**: 
  - Supply Current: 3µA at +1.8V, 6µA at +3.6V, and up to 13µA at +5.5V.

- **Power Consumption**: 
  - Low power consumption of 3µA at +1.8V and 6µA at +3.6V.

- **Operating Temperature Range**: 
  - -40°C to +125°C.

- **Package Type**: 
  - Available in 4-pin SC70 and 6-pin µDFN packages.

- **Special Features**: 
  - Offers a manual reset input with a longer VCC reset timeout period (1120ms or 1200ms) and a shorter manual reset timeout (140ms or 150ms).
  - Reset output state guaranteed valid down to VCC = 1V.
  - Negative-going VCC transient immunity.
  - Various reset output options: Active-Low Push-Pull, Active-High Push-Pull, and Active-Low Open-Drain.

- **Moisture Sensitive Level**: 
  - Moisture Sensitivity Level (MSL) is not explicitly stated in the provided text, but typically, JEDEC J-STD-020E classifications apply to such components.

#### Description:
The MAX6390 is a microprocessor supervisory circuit designed to monitor power supply voltages in low-voltage applications. It ensures that the microprocessor operates correctly during power-up, power-down, and brownout conditions by asserting a reset signal when the supply voltage drops below a predetermined threshold. The device features a manual reset input and various timeout options for the reset signal, making it suitable for applications requiring reliable reset functionality.

#### Typical Applications:
- **Power Management**: Ensures microprocessors are reset correctly during power fluctuations.
- **Portable/Battery-Powered Equipment**: Low power consumption makes it ideal for battery-operated devices.
- **Computers and Controllers**: Used in systems where reliable operation is critical during voltage transitions.
- **Intelligent Instruments**: Provides reset functionality in sensitive measurement and control applications.
- **Dual Voltage Systems**: Monitors multiple voltage levels for system stability.

This component is particularly useful in applications where maintaining the integrity of the microprocessor's operation is essential, especially in environments with fluctuating power supply conditions.
